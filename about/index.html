About me

Hello, my name is Robert Smith and I'm a Front-end Developer, I was born and Grew up in England and I now live in Uppsala, Sweden where I work for wasabiweb, an award winning web-design & conversion agency.

Colophon

This website is hosted directly from my Github repository using Github pages

This website was built on Jekyll A static site genrator, the css is written in Sass using my own custom built framework utilizing the OOCSS philosophies. This framework is also heavily inspired by inuit.css and ITCSS (Both of which are Authored by Harry Roberts.)

I use Grunt.js, a JavaScript task runner to automate repetetive tasks such as Sass compilation, CSS minifcation & Jekyll shell commands.
Grunt runs ontop of Node.js
